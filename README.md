This repository is a bilingual archive of competitive programming problems and solutions from Grader-chan (POSN-style Thai contests).

The goal of this project is to make high-quality Thai and regional contest problems accessible to a global audience by providing:
- Original Thai problem statements
- Accurate English translations
- Solutions in C++


---

## Why This Project Exists

Many excellent competitive programming problems are only available in Thai or local contest platforms.
This project preserves them and makes them readable, searchable, and usable for students around the world.

This is especially useful for:
- Thai students preparing for POSN, COCI, IOI-style contests
- International students looking for new problem sources
- Anyone interested in algorithmic problem solving beyond mainstream judges
