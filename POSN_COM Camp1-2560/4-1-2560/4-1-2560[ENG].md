## Stalactites

A stalagmite grows upward from the ground, and a stalactite hangs downward from above, commonly found in limestone mountains.  
Write a program to print stalactite shapes using the characters `\` and `/` as specified.

The input begins with an integer representing the number of stalactites.  
Each following line gives the length of each stalactite.  
Stalactites are connected side-by-side at their bases.

The number of stalactites is an integer not greater than 20.  
Each stalactite length is an integer not greater than 15.

---

### Input  
- Line 1: An integer, the number of stalactites (≤ 20)  
- Next lines: One integer per line, the length of each stalactite (≤ 15)

---

### Output  
Print the stalactite pattern using `\` and `/` according to the given specification.
