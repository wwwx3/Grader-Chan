## กังหัน 4 ทิศ

กังหัน 4 ทิศรูปแบบหนึ่งที่สร้างจากโปรแกรมคอมพิวเตอร์ จะนำตัวอักษรภาษาอังกฤษมาสร้างเป็นกังหันเรียงกันในแนวทแยงมุม โดยตัวอักษรนี้จะเป็นลำดับของตัวอักษรตามจำนวนเต็มที่ป้อนเข้าไปแล้วลดลำดับลงครั้งละหนึ่งค่า

ตัวอย่างเช่น ถ้าป้อนอินพุตเป็น 3 โปรแกรมจะแสดงผลตัวอักษรเป็น C, B และ A มาเรียงกันเป็นกังหัน โดยให้ตัว A เป็นจุดหมุน  
ถ้าป้อนอินพุตเป็น 5 โปรแกรมจะนำตัวอักษร E, D, C, B และ A มาสร้างเป็นกังหันดังรูป

นอกจากนี้กังหันยังสร้างมาจากเครื่องหมาย `*` ได้เช่นกัน  
หากกังหันทั้งสองรูปแบบวางให้จุดหมุนอยู่ห่างกันเป็นระยะอย่างเหมาะสมของเลขที่ป้อนเข้าไป จะเป็นดังรูป

---

**INPUT**
```

3

```

```

B     B      *

B     B      *
C    C            *

```

**INPUT**
```

5

```

```

### ข้อมูลนำเข้า  

จำนวนเต็มหนึ่งจำนวน **N** โดยที่ \(1 \le N \le 20\)
