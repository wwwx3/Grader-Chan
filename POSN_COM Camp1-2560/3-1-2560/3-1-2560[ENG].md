## Addition in Base x

Write a program to add numbers in base **x**, where **x** ranges from 2 to 16.
The program first receives the base value, then two numbers in that base, and outputs their sum in the same base.

---

### Input

* Line 1: An integer **x** ($(2 \le x \le 16)$), the base
* Line 2: The first number in base **x**
* Line 3: The second number in base **x**

---

### Output

* One line displaying the sum of the two numbers in base **x**

---

### Example

**Input**

```
16
12A
321
```

**Output**

```
44B
```

---

**Input**

```
5
24
32
```

**Output**

```
61
```
