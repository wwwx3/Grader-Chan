## Addition in Base x

จงเขียนโปรแกรมบวกเลขฐาน **x** โดย **x** มีค่าตั้งแต่ 2 ไปจนถึง 16
เมื่อรันโปรแกรมจะให้ป้อนอินพุตค่าแรกเป็นค่าเลขฐานของตัวเลข จากนั้นป้อนตัวเลขเข้าไปสองค่า
โปรแกรมจะแสดงผลบวกของเลขฐานนั้นออกมา

---

### ข้อมูลนำเข้า

บรรทัดแรก เป็นจำนวนเต็ม **x** โดยที่ $(2 \le x \le 16)$
บรรทัดที่สอง เป็นตัวเลขฐาน **x** ตัวที่หนึ่ง
บรรทัดที่สาม เป็นตัวเลขฐาน **x** ตัวที่สอง

---

### ข้อมูลส่งออก

มีบรรทัดเดียว แสดงผลบวกของตัวเลขทั้งสองในเลขฐาน **x**

---

### ตัวอย่าง

**ข้อมูลนำเข้า**

```
16
12A
321
```

**ข้อมูลส่งออก**

```
44B
```

---

**ข้อมูลนำเข้า**

```
5
24
32
```

**ข้อมูลส่งออก**

```
61
```
