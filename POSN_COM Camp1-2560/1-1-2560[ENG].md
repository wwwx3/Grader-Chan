## Prime Numbers Less Than N

A prime number is a positive integer that has exactly two divisors: 1 and itself.  
Given an integer **N** (\(N \le 100\)), print all prime numbers that are strictly less than **N**.

---

### Input  
One line containing an integer **N** $\((N \le 100)\)$

---

### Output  
Print all prime numbers less than **N**, one per line.

---

### Examples  

**Input**
```

10

```

**Output**
```

2
3
5
7

```

**Input**
```

20

```

**Output**
```

2
3
5
7
11
13
17
19

```
