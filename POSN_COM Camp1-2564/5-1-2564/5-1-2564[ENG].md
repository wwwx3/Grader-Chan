## Dice (10 points)

A die is a cube with 6 faces.  
Each face is designed with a specific symbol pattern, as shown in the problem statement.

Write a program that reads a 3-digit number, where each digit represents a face of the die.  
The program must display the symbol representation of each face in order.

The first line contains the 3-digit input.  
The following lines display the dice faces, separated by the `|` character.

---

### Input  
- One line containing a 3-digit integer  
- Each digit represents a face number of the die

---

### Output  
- Display the symbol representation of each die face in order  
- If the input is invalid, output `ERROR`
