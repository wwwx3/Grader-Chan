## Sum of Digits

When a base-10 number is given as input,  
the sum of digits is obtained by adding each digit together.

Write a program that reads a base-10 number on the first line  
and outputs the sum of its digits on the next line.  
The input number contains at most 12 digits.

---

### Input  
- One line containing a base-10 number with at most 12 digits

---

### Output  
- One line displaying the sum of all digits

---

### Example  

**Input**
```

12123

```

**Output**
```

9

```

**Input**
```

123412123213

```

**Output**
```

25

```
